## 2. Exit-gate evidence

- **A level can be authored, playtested, and exported through the editor — entirely headlessly:** *authored* — `EditorState.test.ts` drives paint/place/group/undo/redo as pure snapshot transitions over a `LevelDraft`; *exported* — `exportDraft` strict-parses through the real P2 `parseLevel`, so an invalid draft (out-of-range paint, duplicate id) can never produce a `LevelDefinition`; *playtested* — `Playtest.test.ts` proves interactive `feedInput` drive reproduces a headless `replayTape` of the same frames bit-for-bit, using the sanctioned `CANONICAL_PIPELINE`. All three need zero pixels, so the IRD exit clause is fully satisfiable in P8 with the render/UI share deferred to P9 (dm-0065).
- **Telemetry round-trips into Campaign Intelligence:** `DeathHeatmap.test.ts`'s round-trip test captures a live session, adapts it via `toArchetypeRun` (the dm-0044 substitution — supplying only the archetype label a human capture lacks), assembles a `LevelRecord` whose `run` is telemetry-sourced, and feeds it through the **unmodified** P6 `processCampaign`, asserting a real `CampaignReport` whose `behaviorState.retryCount` equals the capture's reloads. Difficulty-spike detection stays P6's `processCampaign` (dm-0069) — P8 supplies data, not a second algorithm.
- **Death heatmaps are replay-derived, never live-logged (dm-0068):** the heatmap-vs-direct-replay test hand-bins every `defeated`-tick position from an independent `replayObserved` of the same record and asserts `deriveDeathHeatmap` produces the identical tally — recovering deaths from the P1 replay guarantee, with no second capture path to diverge.
- **Immutability held under debug tooling (dm-0070):** `Inspector.setVariable` produces a new committed state through `StateManager.commit()`; the test asserts the prior frozen snapshot is untouched (a `freezeOnCommit` in-place edit would throw) and that the patched state drives the engine deterministically.
- **The determinism axiom is scoped, not violated (dm-0067):** `ToolsIsolation.test.ts`'s wall-clock-confinement scan proves `Date.now`/`performance.now` appear only under `tools/profiler/`; `src/core|systems|components|entities|schema|eval|gen` stay wall-clock-clean, and profiler output never enters `WorldState` or a replay-checked value.
- **`tools/` is isolated and consumes `src/` only through public seams (dm-0066/dm-0074):** `ToolsIsolation.test.ts` proves nothing under `src/` imports `tools/`; `tools/` never imports `src/gen/` (P8 has zero P7 dependency), never imports `eval/local` audits, `AgentPolicy`/`Evaluate`/`EmergentFun`, or gdos scoring-gate internals; it reaches `eval/` only via `AgentHarness` (`CANONICAL_PIPELINE`/`replayObserved`) and `campaign/` (`analyzeTape`/`processCampaign`), and reads gdos OUTPUT types (`ArchetypeRun`) and `ArchetypeName` as types only. Confirmed by git: **zero modifications under `src/eval/` (campaign/ included) or `src/gen/`** — the verified P4/P6/P7 subtrees were consumed as read-only libraries.
- **No rendering creep (dm-0065):** the scan proves no `document`/`window`/`getContext`/WebGL/canvas API anywhere under `tools/` — every editor/overlay/telemetry output is plain data; the presentation share is P9's.
- **No content authored:** every `LevelDefinition` P8 touches is test-scoped scaffolding (grid fixtures, blank editor drafts); there is no `levels/` directory and no chapter authoring — the content gate holds through P8 close (P10 authors real content, now unlocked by M3).
